Brief Description of the System
Smart Learning is a modern, responsive web-based student platform designed for high schools.
The system supports three user roles:

Students – Can view their personal performance, assignments, and discipline reports.
Teachers – Can manage students, update academic performance (with scores and remarks), create assignments with resource
Parents – Can view their child’s academic progress and discipline records.

Key Features:

Secure user registration and login
Assignment creation and submission system
Discipline reporting and tracking

The platform is built using HTML, Tailwind CSS, and JavaScript, with data stored locally using LocalStorage, making it lightweight and easy to run.
